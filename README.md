# 主食碳水查詢與血糖紀錄

個人用的離線血糖管理工具：常見主食碳水/GI/GL 查詢、份量計算、血糖紀錄、收藏、匯出備份。可用瀏覽器「加到主畫面」當成 PWA App 使用。

## 功能

- **查詢**：關鍵字、分類、排序（碳水/GI/GL）、升糖速度篩選、收藏置頂、搜尋字高亮
- **份量計算器**：每張卡片可輸入「實際吃幾 g」，即時算出當餐碳水與 GL，記住上次輸入
- **GL（升糖負荷）**：每筆食物顯示每 100g 與當前份量的 GL，比單看 GI 更貼近實際血糖負擔
- **收藏 (★)**：常吃的食物標星，可切換「只看收藏」並自動置頂
- **血糖紀錄**：每張卡片可新增結構化紀錄（日期、餐別、份量、餐前/餐後血糖、備註），自動算 Δ
- **紀錄分頁**：以日期分組瀏覽所有紀錄、最近 7 天統計、平均餐後 Δ
- **工具分頁**：主題切換（自動/亮/暗）、匯出 JSON 備份、匯入還原、清除資料、依升糖速度看實際 Δ 平均
- **PWA**：離線可用、可加到主畫面當 App
- **暗色模式**：跟隨系統或手動指定

## 資料口徑

- 碳水：以熟食或可直接食用重量的每 100g 總碳水為主
- 升糖速度：以 GI 分類，低 GI ≤55、中 GI 56-69、高 GI ≥70
- GL = GI × 碳水 / 100，把份量也納入考量（>20 為高、11-19 中、≤10 低）
- 台灣主餐、便當、漢堡、夜市小吃、手搖飲多為混合餐；若沒有公開 GI，會以主要碳水來源估算，並在備註寫明整份碳水範圍
- 水果以新鮮可食部每 100g 為主；果汁、果乾、罐頭糖水水果不套用同一筆數值
- 所有資料只存在使用者自己的瀏覽器 localStorage，不會上傳或同步；建議定期透過工具分頁匯出備份

## 食物分類

- 米飯、麵 / 粉、麵包、穀物、玉米 / 其他、根莖、豆類、水果
- 台灣主餐（便當、雞排、漢堡等）
- **早餐店**（蛋餅、蘿蔔糕、燒餅、油條、蔥抓餅、厚片吐司、三明治、飯糰、豆漿、米漿）
- **夜市小吃**（刈包、肉圓、蚵仔麵線、臭豆腐、地瓜球、紅豆餅、胡椒餅、香腸、雞蛋糕、潤餅）
- **麵食湯品**（陽春麵、餛飩麵、麻醬麵、擔仔麵、米粉湯、肉羹麵、酸辣湯、四神湯、鹹粥）
- **台式飯類**（油飯、肉粽、咖哩飯、燴飯、海南雞飯、地瓜飯）
- **手搖飲**（珍奶、加糖紅綠茶、冬瓜茶、木瓜牛奶、楊桃汁、青草茶、無糖茶）
- **台式甜點**（豆花、愛玉、芒果冰、紅豆湯、湯圓、麻糬、芋圓、黑糖剉冰）
- **便利商店**（御飯糰、茶葉蛋、關東煮料）

## 本機開啟

PWA / service worker 需要 http(s) 才能註冊，建議用本機伺服器：

```bash
python -m http.server 8080
```

然後開瀏覽器到 http://localhost:8080/。直接雙擊 `index.html` 也能用，只是不會有離線快取。

## 備份

工具分頁 → 匯出我的資料 → 下載 JSON。換手機或清快取前記得匯出，到新環境用「匯入備份」還原。

## 檔案結構

- `index.html`、`styles.css`、`app.js`：主要程式
- `manifest.json`、`sw.js`、`icon.svg`：PWA 設定與快取
